## Run Locally

Clone the project

```bash
  git clone https://github.com/Cyteon/convoke
```

Go to the project directory

```bash
  cd convoke
```

Rename `config.example.yaml` to `config.yaml` and populate the values

Start the server

```bash
  go run .
```
